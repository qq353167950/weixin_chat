# 更新日志

每个版本一段，发版时 CI 自动提取对应版本段落作为 Release 说明。
格式约定：每条以 `- ` 开头，用「新增/修复/调整/优化」开头一句话说清，不写实现细节。

## v1.9.4

- 修复：更新重启后弹出「Failed to remove temporary directory」错误框的问题——启动新版时不再占用旧版临时目录（重要，本次更新后的下一次更新起彻底生效）

## v1.9.3

- 修复：更新到新版后界面功能仍是旧版、必须手动关掉重开才生效的问题——更新后自动重启即为最新版（重要，本次更新后的下一次更新起彻底生效）

## v1.9.2

- 修复：设置页「查本机出口 IP」点击报连接错误（连接被重置）——改为多个查询源自动轮换，国内网络也能查到（重要）
- 修复：生成封面时进度日志被覆盖，「总结文章 XX 字」等中间过程一闪而过——现在完整保留在日志里，全程可见
- 修复：内置更新下载失败后自动打开浏览器会影响默认浏览器登录状态——改为用系统默认方式直接打开发布页，不动浏览器任何数据
- 优化：设置页「查本机出口 IP」按钮样式与「测试公众号连接」统一
- 优化：生图尺寸自动适配——切换生图方式（OpenAI/万相）后即使尺寸格式没改也能自动纠正，并在设置页标注正确格式

## v1.9.1

- 优化：封面改为「先读文章总结主题、再定制生图提示词」，一图一议贴合本文，告别每篇都一个套路的固定风格
- 优化：封面允许画面出现文字，可生成带标题感的封面
- 说明：封面提示词现在会用写作模型总结文章（会用到写作模型），失败时自动退回通用提示词

## v1.9.0

- 调整：配图改为「写作时就地生成」——写文章的同时就定好每张图的位置和内容，图文一次生成、贴合正文，不再是写完再回头硬贴
- 优化：配图位置更准，告别过去「找不到插图点位就跳过」的情况
- 优化：正文配图允许出现文字标签，流程图/结构图可以带上关键词说明
- 优化：小标题若自带序号（第一节、一、、1. 等）不再重复叠加数字徽章，排版更干净
- 优化：正文引用金句卡片改为全圆角，观感更柔和

## v1.8.0

- 新增：设置页「连接测试」——一键测试写作模型和公众号配置是否正确，不用跑完整流程才发现配错
- 新增：设置页「查本机出口 IP」，点击结果即复制，方便去公众号后台加白名单
- 新增：编辑器本地草稿兜底——程序崩溃或断电后重新打开可恢复未保存的内容
- 新增：发布页标题/摘要/作者实时字数显示（接近上限自动变色提醒）
- 优化：检查更新结果缓存 1 小时，频繁启动不再受 GitHub 接口限流影响
- 优化：文章结尾不再千篇一律的「今天就能动手」行动号召，随内容自然收束
- 优化：更新切换新版本时先收起旧窗口再打开新窗口，衔接更自然

## v1.7.5

- 修复：应用内更新彻底重做——不再调用安装器和命令行窗口，程序直接替换自身文件后自动重启，全程无任何弹窗（重要，本次更新后生效）

## v1.7.4

- 修复：点「开始搜索选题」等按钮无响应、提示请求超时的问题（重要，建议尽快更新）
- 修复：技术文中 arr[1] 这类代码数组下标被误删的问题
- 修复：文中含彩色字体时摘要出现乱码片段的问题
- 修复：含中文的链接被排版空格打断失效的问题
- 修复：行内代码里演示的 HTML 代码被错误渲染的问题
- 修复：「一键去 AI 味」进行中刷新页面进度丢失的问题
- 修复：任务运行中保存文章可能互相覆盖的问题
- 优化：命令行模式输入容错与失败后可直接改手动输入主题

## v1.7.3

- 修复：应用内更新不再需要权限确认——安装改为用户级（同 Chrome/VS Code 模式），装到用户目录，安装与更新全程无弹窗
- 说明：从旧版（装在 Program Files）升级到本版时，最后一次更新仍可能弹一次权限确认；本版装好后以后彻底免打扰

## v1.7.2

- 优化：候选选题标题更抓人——短小有冲击力（14-24 字），突出主题可适度夸张，不再是格式化长标题
- 优化：写作参考默认从 3 篇增加到 5 篇、单篇学习内容加长，多篇取长补短，成文更自然
- 调整：设置页各控件统一高度对齐，排版更整齐

## v1.7.1

- 修复：装在 Program Files 时应用内更新报「错误码 5」失败的问题（更新时会弹一次系统权限确认，点「是」即可）
- 新增：升级后自动把新版本新增的配置合并进你的 .env，原有配置一律不动
- 调整：设置页画面风格、排版主题、搜索服务、生图方式全部显示中文名称

## v1.7.0

- 优化：安装包构建不再依赖外网下载语言文件；卸载时清理缓存但保留配置与文章
- 修复：Mac 版「关于」显示错误版本号的问题
